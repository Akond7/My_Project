## 1. Игровые механики

### 1.1 Основная петля (Core Loop)


```

[Старт ночи] → [Проверка камер / дверей] → [Перемещение врагов] → [Трата энергии на защиту] → [Победа (6 AM) / Смерть]

```

Игрок начинает смену и следит за мониторами. Враги периодически меняют локации, приближаясь к комнате охраны. Игрок реагирует на угрозу, закрывая двери или включая свет, расходуя при этом ограниченный запас энергии. Цикл продолжается до истечения таймера (утро) или до проникновения врага.

### 1.2 Управление

| Кнопка / Ввод | Действие |
|---------------|----------|
| `ЛКМ` | Взаимодействие с UI (переключение камер, кнопки света, дверей) |
| `Esc` | Пауза |

### 1.3 Механика движения

- **Скорость:** 0 px/с (игрок статичен, находится в кресле охранника)
- **Гравитация:** Отсутствует
- **Прыжок:** Отсутствует
- **Ограничения:** Взаимодействие происходит исключительно через курсор мыши и элементы интерфейса на экране
- **Механика движения противников:** Алгоритм в разработке

### 1.4 Боевая механика / Основное взаимодействие

Игрок не может атаковать врагов. Основной конфликт строится на контроле ресурсов (энергия) и внимания.

- **Нанесение урона:** Игрок не наносит урон. Враги бессмертны.
- **Получение урона:** Мгновенная смерть (скример), если враг добирается до открытой двери комнаты охраны.
- **Расход энергии:** Пассивное потребление (медленно), включенный монитор/свет/закрытая дверь (быстро). Если энергия падает до 0, защита отключается.
- **Условие победы:** Продержаться до конца таймера ночи (условные 4–6 минут реального времени).
- **Условие поражения:** Враг проникает в комнату или заканчивается энергия в критический момент.

### 1.5 Прогрессия и нарастание сложности

| Этап | Условие | Изменение |
|------|---------|-----------|
| Старт | 12:00 AM | Базовые параметры, враги двигаются редко |
| Середина ночи | 03:00 AM | +50% к скорости перемещения врагов, появляются на ближних камерах |
| Конец ночи | 05:00 AM | Максимальная агрессия врагов, запас энергии игрока на исходе |

## 2. Игровые сущности

### 2.1 Игрок

| Параметр | Значение |
|----------|----------|
| HP / Жизни | 1 (Смерть с одного удара) |
| Энергия | 100% |
| Урон | 0 |
| Спрайт | Отсутствует (вид от первого лица) |

### 2.2 Враги / Препятствия

| Тип | HP | Скорость | Урон | Поведение | Спрайт |
|-----|----|----------|------|-----------|--------|
| Враг 1 | Бессмертен | Перемещение раз в X сек | instant kill | Движется по камерам к левой двери | `enemy_1.png` |
| Враг 2 | Бессмертен | Перемещение раз в Y сек | instant kill | Движется по камерам к правой двери / слепой зоне | `enemy_2.png` |

### 2.3 Предметы / Бонусы

| Предмет | Эффект | Длительность | Спрайт |
|---------|--------|--------------|--------|
| Отсутствуют | В игре нет собираемых бонусов | — | — |

### 2.4 Уровни / Окружение

- **Количество уровней:** 1 комната охраны + 3–4 статичных экрана камер
- **Тип генерации:** Фиксированные 2D-фоны
- **Размер уровня:** Размер окна игры (например, 1280×720 px)
- **Тайлсет:** Не используется (используются цельные изображения экранов)

## 3. Интерфейс (UI / UX)

### 3.1 Схема экранов


```

[Главное меню]
      │
      ├─── [Настройки]
      │     
      ├─── [Рекорды]  
      │
      └─── [Игра] ──── [Пауза] ──── [Главное меню]
                │
                └─── [Game Over] ──── [Главное меню]
                                 └─── [Играть снова]

```

### 3.2 Главное меню

*Вайрфрейм (Wireframe, схематичный набросок интерфейса):*


```

┌──────────────────────────────┐
│      НОЧНАЯ СМЕНА ОХРАНЫ     │
│                              │
│        [ Играть     ]        │
│        [ Настройки  ]        │
│        [ Рекорды  ]          │
│        [ Выход      ]        │
│                              │
│                              │
└──────────────────────────────┘

```

### 3.3 HUD (Heads-Up Display) — постоянно видимый игровой интерфейс

*Вайрфрейм (Комната охраны):*


```

┌──────────────────────────────────┐
│ Энергия: 85%         Время: 2 AM │
│                                  │
│ [Дверь] [Свет]    [Свет] [Дверь] │
│                                  │
│       [ Открыть монитор ]        │
└──────────────────────────────────┘

```

| Элемент | Позиция | Описание |
|---------|---------|----------|
| Энергия | Верх-лево | Полоска или процент оставшейся энергии |
| Время | Верх-право | Текущее время (от 12 AM до 6 AM) |
| Кнопки дверей/света | По бокам | Интерактивные зоны для управления защитой |
| Кнопка монитора | Низ-центр | Открывает/закрывает планшет с камерами |

### 3.4 Экран паузы


```

┌──────────────────────────────┐
│            ПАУЗА             │
│                              │
│      [ Продолжить   ]        │
│      [ Главное меню ]        │
└──────────────────────────────┘

```

### 3.5 Экран Game Over


```

┌──────────────────────────────┐
│         GAME OVER            │
│                              │
│  Вы продержались до: 4 AM    |
│                              | 
│      [ Играть снова ]        │
│      [ Главное меню ]        │
└──────────────────────────────┘

```
