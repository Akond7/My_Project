На текущем этапе реализован технический каркас проекта:

- структура проекта;
- запуск игры через `main.py`;
- окно `pygame`;
- базовая машина состояний;
- главное меню;
- заглушка игрового экрана;
- логирование;
- тесты через `pytest`;
- проверка кода через `ruff`.

---

## 1. Используемые технологии

В проекте используются следующие инструменты и библиотеки:

- **Python 3.12** — основной язык разработки;
- **uv** — управление зависимостями и запуск проекта;
- **pygame** — создание игрового окна, обработка событий и отрисовка;
- **pytest** — автоматическое тестирование логики проекта;
- **ruff** — проверка качества и стиля кода;
- **logging** — запись событий игры в лог-файл.

## 2. Структура проекта

Проект использует структуру `src-layout`.

```text
project/
├── assets/                         # игровые ресурсы
├── docs/                           # документация проекта
│   ├── GDD.md
│   ├── Концепт-документ.md
│   └── TDD.md
├── src/
│   └── night_security_shift/
│       ├── __init__.py
│       ├── app.py                  # главный класс приложения
│       ├── settings.py             # настройки проекта
│       ├── states/
│       │   ├── __init__.py
│       │   ├── menu_state.py       # главное меню
│       │   └── game_state.py       # игровой экран-заглушка
│       └── utils/
│           ├── __init__.py
│           └── logger.py           # настройка логирования
├── tests/
│   ├── test_logger.py              # тест логирования
│   └── test_states.py              # тесты переходов состояний
├── main.py                         # точка входа
├── pyproject.toml                  # настройки проекта и зависимости
├── uv.lock                         # зафиксированные версии зависимостей
└── .gitignore
